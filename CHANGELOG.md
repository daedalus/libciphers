# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

### Added
- Initial release with classical cipher implementations
- Shift ciphers: Caesar, ROT13, ROT-N, Atbash, Affine
- Polyalphabetic ciphers: Vigenère, Beaufort, Autokey, Running Key, Porta, Gronsfeld
- Transposition ciphers: Columnar, Rail Fence, Scytale
- Digraph ciphers: Playfair, Four-Square
- Fractionated ciphers: Bifid, Trifid
- Hill cipher (2x2 matrix)
- Enigma (3-rotor simplified)
- Statistical analysis functions

[0.1.0]: https://github.com/daedalus/libciphers/releases/tag/v0.1.0
